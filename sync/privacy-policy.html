プライバシーポリシー

SyncMix 及び SyncMatch （以下、「当アプリ」といいます。）は、GDGoC Japan Hackathon 2026 の開発チーム「わらび餅キング」（以下、「開発チーム」といいます。）が提供しています。当アプリは、ユーザーの皆様の個人情報およびプライバシーの保護に努めております。本プライバシーポリシーでは、当アプリが収集する情報、その利用目的、および情報の取り扱いについて説明します。

1. 取得する情報

当アプリは、サービスの提供にあたり以下の情報を取得・利用します。

  • Googleアカウント情報（ユーザーID、氏名、メールアドレス、プロフィール画像など）
  • YouTube Data APIを通じて取得する情報（ユーザーが高評価した動画、および検索した動画についてユーザーが指定した名前でプレイリストを追加するための情報）

2. 情報の利用目的とデータの保存

取得した情報は、以下の目的で利用・保存されます。

  • 当アプリへのログインおよびユーザー認証のため
  • ユーザーの感性や好みを分析し、最適な作品を提案する機能（ベクトル化やマッチング等）を提供するため
  • 当アプリの機能改善、不具合対応、および利便性向上のため

取得した高評価動画の情報は、動画の概要欄とコメントの内容に基づいてエンベディング処理（ベクトル化）された上で、当アプリのデータベースに保存されます。

3. 第三者サービスおよびAPIの利用について

当アプリは、機能提供のために以下の外部APIを利用しており、それぞれの規約に基づいてデータが処理されます。

① YouTube API サービスの利用
当アプリは、YouTube API Servicesを利用してデータを取得しています。当アプリを利用するユーザーは、YouTube利用規約（https://www.youtube.com/t/terms）に拘束されることに同意したものとみなされます。
また、ユーザーの情報はGoogleプライバシーポリシー（http://www.google.com/policies/privacy）に基づいて取り扱われます。

② 生成AI（Gemini API）の利用
当アプリは、取得したテキストデータ（動画情報やコメント等）の分析・要約のために、Google LLCが提供するGemini APIを利用しています。取得した高評価動画の情報やユーザーの検索クエリが、AIモデルの学習に使用されることは一切ありません。

4. データの削除とデータアクセスの取り消し

データの削除について
データベースに保存されたご自身のデータの削除を希望される場合は、下記のお問い合わせ先に記載されたデベロッパーのメールアドレス宛に、データの削除を希望する旨とご自身の「ユーザーID」を送信いただくことで対応可能です。

アクセス権の取り消し
ユーザーは、Googleアカウントのセキュリティ設定（https://myaccount.google.com/permissions）から、当アプリに対するデータへのアクセス権をいつでも取り消すことができます。

5. 免責事項

当アプリの利用により生じたトラブルや損害について、開発チームは一切の責任を負いかねます。あらかじめご了承ください。

6. プライバシーポリシーの変更

当アプリは、必要に応じて本プライバシーポリシーを変更することがあります。変更後のプライバシーポリシーは、本ページに掲示された時点から効力を生じるものとします。
